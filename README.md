Este proyecto consiste en implementar un simulador de batallas tipo RPG (Rol-Playing Game o juego basado 
en roles) en Java utilizando únicamente la consola. El jugador elegirá un personaje con ciertas características 
y estadísticas de rendimiento y se enfrentará a enemigos controlados por el sistema en una batalla por turnos. 
Cada personaje tiene atributos de rendimiento como salud, ataque y defensa. Los enemigos tendrán 
comportamientos distintos según su tipo. Este proyecto permite practicar conceptos fundamentales de 
programación orientada a objetos como la herencia, polimorfismo, encapsulamiento y sobreescritura de 
métodos, así como el diseño de ciclos de juego y administración de estados.
